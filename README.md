# Taxi Income Analysis

This project analyzes the income of taxi drivers using data analytics.

## Goal
To understand drivers' performance and compare top earners with average drivers.

## What is included
- Average and median income  
- Top 10% highest earning drivers  
- Comparison between top drivers and regular drivers  

## Tools
Python, NumPy, and data analysis techniques.

## Result
The analysis provides insights into income distribution and helps support business decisions such as bonuses and performance evaluation.
